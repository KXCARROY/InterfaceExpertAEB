Création d'un environnement virtuel avec Django

python -m venv project-name

Cela va créer un dossier nommé « project-name » s’il n’existe pas déjà et va configurer l’environnement virtuel. 
Pour activer l’environnement, exécutez

project-name\Scripts\activate.bat

Télécharger Django

python -m pip install Django

Puis pour lancer le serveur virtuel 

python manage.py runserver

Ce qui va lancer le site sur 127.0.0.1:8000
